# Simple Linear Regression using Strava Data

## 📌 Project Overview
This project demonstrates simple linear regression using my running data from Strava. The goal is to predict running distance based on moving time.

## 📊 Dataset
The dataset contains running activities with features like:
- Moving Time
- Distance

## 🧠 Model Used
Simple Linear Regression (Scikit-learn)

## ⚙️ Steps Performed
- Data preprocessing
- Feature selection
- Model training
- Prediction
- Visualization

## 📈 Results
The model shows a linear relationship between moving time and distance.

## 🛠️ Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn
